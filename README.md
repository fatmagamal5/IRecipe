# IRecipe - A Modern Recipe Sharing Platform

IRecipe is a dynamic and feature-rich web application designed for food enthusiasts to discover, share, and manage their favorite recipes. It offers a seamless user experience with a modern interface and robust backend, making it the perfect platform for all your culinary needs.

## Key Features

*   **User Authentication**: Secure user registration and login with JWT and session-based authentication.
*   **Recipe Management**: Create, edit, and delete your own recipes with a user-friendly interface.
*   **Admin Dashboard**: A powerful admin panel for managing users, recipes, and reviews, with detailed statistics and moderation tools.
*   **Review System**: Users can leave reviews and ratings on recipes, with admin approval for quality control.
*   **Favorites**: Save your favorite recipes for easy access later.
*   **Responsive Design**: A beautiful and responsive design that works on all devices.

## Tech Stack

*   **Backend**: Node.js, Express.js, MongoDB
*   **Frontend**: EJS, Bootstrap, JavaScript
*   **Authentication**: JWT, bcryptjs, express-session
*   **Database**: Mongoose
*   **File Uploads**: Multer
